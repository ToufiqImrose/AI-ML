Neural Network Classification Project
Overview

This project compares two neural network models — Shallow Neural Network and Deep Neural Network — using the Heart Disease dataset. The goal is to predict whether a patient has heart disease and compare the performance of both models.

Dataset
Heart Disease Dataset (.csv)
Binary classification problem

Project Steps
Data preprocessing (missing values, encoding, scaling)
Train-test split (80/20)
Build Shallow Neural Network (1 hidden layer)
Build Deep Neural Network (3 hidden layers + Dropout)
Hyperparameter tuning
Model evaluation

Training history (Loss & Accuracy)

Results

Both models achieved excellent performance with:

Accuracy: 100%
Precision: 100%
Recall: 100%
F1-Score: 100%
AUC: 1.00

The shallow model trained faster, while the deep model used more layers but gave similar results.
